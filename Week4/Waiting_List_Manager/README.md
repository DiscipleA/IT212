# Waiting_List_Manager
 Assignment IT212: Waiting List Program

This Java program demonstrates how to manage a customer waiting list for a service using Queue and Set interfaces.

🚀 How It Works

- Manages a waiting list using a Queue and Set.
- Prevents duplicate entries by checking before adding a person.
- Offers functionalities to add, serve, check presence, show size, and display the waiting list order.

🛠 Features

- Implementation of a Queue and Set using LinkedList and HashSet respectively.
- Adding a person to the waiting list and checking for duplicates.
- Serving the person at the front of the waiting list.
- Checking if a person is in the waiting list.
- Retrieving the size of the waiting list.
- Displaying the waiting list in the order of service.

▶️ How to Use

- Clone this repository or download the .java file.
- Compile and run the program:

```
javac WaitingListManager.java
java WaitingListManager.java

```
