# Geometry_App
 Assignment IT212: Simple Geometry Application using Abstract Classes

This is a simple Java program that shows the functionality of Inheritance & Polymorphism in Java, and demonstrates object oriented Ingeritances using a Geometrical shapes scenario.

🚀 How It Works

The Geometric Shapes Calculator program allows you to calculate the area and perimeter of different shapes like Circle, Rectangle, and Triangle. Each shape is represented by a subclass of the abstract class Shape, implementing methods to calculate the area and perimeter based on shape-specific properties.

🛠 Features

- Abstract class `Shape` with calculateArea() and calculatePerimeter() abstract methods.
- Concrete method in `Shape` returning the name of the shape.
- Subclasses Circle, Rectangle, and Triangle with specific properties and formulas for area and perimeter calculations.
- Main class creating instances of each shape and displaying their name, area, and perimeter.

▶️ How to Run

- Clone this repository.
- Make sure that all the class/object files in the same folder.
- Adjust radious, length, width, etc..., parameters in Main as desired. 
- Compile the program:

```
javac Main.java
java Main.java
```

Area: 78.54
Perimeter: 31.42
