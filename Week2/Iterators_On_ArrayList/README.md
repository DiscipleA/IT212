# Iterators_On_ArrayList
Assignment IT212: Apply Iterators on ArrayList

This Java program demonstrates how to manipulate and iterate through an ArrayList of Integers.

🚀 How It Works

- An ArrayList of Integers is created and initialized with ten random integers.
- The program uses an Iterator to traverse the list, printing each integer, and calculating the sum of the integers.
- After the iteration completes, the total sum of the integers is displayed.

🛠 Features

- Populates an ArrayList with ten random integers.
- Utilizes Iterator to iterate over the ArrayList and calculate the sum of the integers.
- Demonstrates adding and removing items from the list during iteration using Iterator's `remove()` method and list's `add()` and `remove()` methods.

▶️ How to Run

- Clone this repository or download the .java file.
- Compile and run the program:

```
javac Main.java
java Main
```

📌 Example Output

```
[4, 6, 3, 3, 6, 6, 6, 6, 7, 3]
```
